●Stress Engineer
FEA, Stress Analysis & Failure Criteria
Multi-load-case structural analysis of bracket components using Abaqus and HyperWorks

Boundary Conditions & Contacts

  • RBE2 rigid elements for load distribution (vs. RBE3 interpolation comparison)
  • Friction contact: Ti–Fe μ_s = 0.4, μ_k = 0.3
  • Friction contact: Fe–Fe μ_s = 0.5, μ_k = 0.4
  • 3 contact pairs: bracket–bearing, safety ring–bracket, safety ring–bearing

Failure Criteria Implemented

  • von Mises — isotropic yielding baseline
  • Drucker-Prager — pressure-dependent yield (tension-compression asymmetry)
  • Hill 48 — anisotropic plasticity for AM builds
  • Cuntze-Bold — composite-adapted failure for CFRP phases

Mesh Quality Criteria

  • Warpage: 0 elements failed
  • Aspect ratio: <130 elements (<0.01%)
  • Skew: <310 elements (<0.01%)
  • Tet collapse: 0 failed
  • Orthogonality: ≤75 elements

●Materials Engineer
Ti6Al4V AM Material Validation
FEM correlation study against destructive coupon testing — characterising anisotropy from additive manufacturing build directions

Testing Standards Applied

  • ASTM E8 — standard tension test (flat specimens)
  • ASTM B831 — shear test for AM flat specimens
  • ASTM E9 — compression testing
  • HSB — internal aerospace standard (DIN/LN-based)

Key Findings — Anisotropy

  • Build direction stiffness: +7% vs XY plane (tension & compression)
  • Build direction Rp0.2: +6% — stronger in build axis
  • Compression stiffness: +10.9% (107.2 GPa → 118.9 GPa)
  • Compression yield: +11.8% (788 MPa → 868 MPa)

●Manufacturing / AM Engineer
Additive Manufacturing Constraints & CFRP Connection
Integrating AM design rules into topology optimisation and designing structural interfaces between AM-Ti and CFRP beams

AM Design Constraints in Optimisation

  • Minimum member thickness enforced (SLM resolution limits)
  • Overhang angles restricted to avoid support structures
  • Build direction specified: vertical Z-axis
  • Density filter (SIMP method) with manufacturing penalisation
  • Post-optimisation interpretation for printability

AM Bracket — HiWi HAP3 Project

  • Bracket redesign for HAP3 solar aircraft system
  • Story-based redesign: iterative improvement cycle
  • Multiple hinge and bracket variants evaluated (HAP3_hinges, HAP3_brackets)
  • Mass reduction target achieved through TO + AM manufacturing

CFRP–AM Bolt Connection Study (Feb 2023)

Designed and analysed the structural interface between a carbon fibre reinforced polymer (CFRP) beam and an additively manufactured titanium connector. Investigated load transfer mechanisms through bolted joints.

  • Bolt pre-load and clamping force analysis
  • Bearing stress, shear-out, net section failure modes checked
  • Contact modelling between dissimilar materials (CFRP laminate + Ti SLM)
  • Validated against analytical calculations (Bruhn method)
